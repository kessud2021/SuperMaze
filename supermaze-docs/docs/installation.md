# Installation

## Requirements
- Modern web browser
- Optional: Node.js for WebSocket server and leaderboard
- Python + MkDocs if using local documentation

## Steps
1. Clone the repository:
```bash
git clone https://github.com/kessud2021/SuperMaze
cd SuperMaze
```

Open `index.html` in a browser to run the game.

Optional: Run WebSocket Server
```
cd server
npm install
node server.js
```
