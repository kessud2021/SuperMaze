# Scripts

## script.js Overview
### Initialization
- Loads maze layout
- Sets player starting position
- Binds keyboard events

### Event Handling
- `keydown` listens for WASD inputs
- Validates movement against walls

### Maze Rendering
- Renders maze using canvas or DOM elements
- Updates player position in real-time
- Refreshes display each frame

### Reset Functionality
- Resets player position
- Optionally regenerates the maze
